Detailed Description

RAII worktree directory swapper.

Swaps filesystem root_dir, bash working_dir, and git repo_dir to the worktree path. Restores all on destruction (success or failure). Uses entropic_mcp_server_set_working_dir() per-server via the engine's server access callback.
