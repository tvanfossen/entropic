## Scope

First real C++ library code: `librentropic-config.so`. After v1.8.1, the
engine can load YAML config files, validate them, resolve bundled model
paths, parse identity/constitution/app_context prompt files, and build
classification prompts for the router.

This is a faithful port of:
- `src/entropic/config/schema.py` -- all config structs + validators
- `src/entropic/prompts/__init__.py` -- frontmatter parsing + prompt builder
- Bundled data file discovery and resolution

v1.8.0 defined config STRUCTS in `include/entropic/types/config.h` (with
defaults). v1.8.1 adds the LOADER that populates those structs from YAML
and the VALIDATION layer that enforces constraints.

No inference, no MCP, no engine loop. Pure config + prompt infrastructure.

## Implementation Plan

### Phase 1: ryml Integration + YAML Parser Utilities

ryml (`ryml_all.hpp`) is already vendored in `extern/ryml/` by v1.8.0.
Build the extraction utility layer that all config/prompt code uses.

`src/config/yaml_util.h` -- internal helpers (not in public headers):

```cpp
/// @file yaml_util.h
/// @brief ryml extraction helpers for config parsing.
///
/// Wraps ryml's tree API with type-safe extraction functions that
/// log warnings for unknown keys and provide clear error messages
/// for type mismatches.

#pragma once

#include <ryml_all.hpp>
#include <filesystem>
#include <optional>
#include <string>
#include <vector>

namespace entropic::config {

/// @brief Extract a string value from a YAML node.
/// @param node The ryml node to extract from.
/// @param key The key to look up.
/// @param out Output string. Unchanged if key is absent.
/// @return true if key was found and extracted.
bool extract(ryml::ConstNodeRef node, c4::csubstr key, std::string& out);

/// @brief Extract an int value from a YAML node.
bool extract(ryml::ConstNodeRef node, c4::csubstr key, int& out);

/// @brief Extract a float value from a YAML node.
bool extract(ryml::ConstNodeRef node, c4::csubstr key, float& out);

/// @brief Extract a double value from a YAML node.
bool extract(ryml::ConstNodeRef node, c4::csubstr key, double& out);

/// @brief Extract a bool value from a YAML node.
bool extract(ryml::ConstNodeRef node, c4::csubstr key, bool& out);

/// @brief Extract a filesystem path with ~ expansion.
bool extract_path(ryml::ConstNodeRef node, c4::csubstr key,
                  std::filesystem::path& out);

/// @brief Extract an optional path (null/absent = nullopt).
bool extract_optional_path(ryml::ConstNodeRef node, c4::csubstr key,
                           std::optional<std::filesystem::path>& out);

/// @brief Extract a tri-state path (null = nullopt, false = disabled, string = path).
/// Maps to Python's TriStatePath: None | False | Path.
/// YAML values: absent/null = nullopt, false = disabled, string = path.
bool extract_tri_state_path(
    ryml::ConstNodeRef node, c4::csubstr key,
    std::optional<std::filesystem::path>& out, bool& disabled);

/// @brief Extract a vector of strings from a YAML sequence node.
bool extract_string_list(ryml::ConstNodeRef node, c4::csubstr key,
                         std::vector<std::string>& out);

/// @brief Expand ~ to home directory in a path.
std::filesystem::path expand_home(const std::filesystem::path& p);
```

### Phase 2: BundledModels Registry

Port `_load_bundled_models()` and `resolve_model_path()` from `schema.py`.

`include/entropic/config/bundled_models.h`:

```cpp
/// @file bundled_models.h
/// @brief Bundled model registry — resolves keys to paths.

#pragma once

#include <filesystem>
#include <string>
#include <unordered_map>

namespace entropic::config {

/// @brief Entry in the bundled model registry.
///
/// Maps a key (e.g., "primary") to a model name, URL, and metadata.
/// Parsed from bundled_models.yaml.
struct BundledModelEntry {
    std::string key;              ///< Registry key (e.g., "primary")
    std::string name;             ///< Model filename stem (e.g., "Qwen3.5-35B-A3B-UD-IQ3_XXS")
    std::string url;              ///< Download URL
    double size_gb = 0.0;         ///< Model size in GB
    std::string adapter;          ///< Adapter name (e.g., "qwen35")
    std::string description;      ///< Human-readable description
};

/// @brief Bundled model registry loaded from bundled_models.yaml.
///
/// @par Resolution
/// When a config field contains a registry key (e.g., "primary"),
/// resolve() returns ~/models/gguf/{name}.gguf. When the value is
/// already a path, it passes through with ~ expansion.
///
/// @par Thread safety
/// Immutable after load. Safe to read from any thread.
class BundledModels {
public:
    /// @brief Load registry from YAML file.
    /// @param path Path to bundled_models.yaml.
    /// @return Empty string on success, error message on failure.
    std::string load(const std::filesystem::path& path);

    /// @brief Check if a key exists in the registry.
    bool contains(const std::string& key) const;

    /// @brief Get entry by key.
    /// @return Pointer to entry, or nullptr if not found.
    const BundledModelEntry* get(const std::string& key) const;

    /// @brief Resolve a model reference to a filesystem path.
    ///
    /// If value matches a registry key, returns ~/models/gguf/{name}.gguf.
    /// Otherwise treats value as a direct path with ~ expansion.
    ///
    /// @param value Registry key or direct path string.
    /// @return Resolved filesystem path.
    std::filesystem::path resolve(const std::string& value) const;

    /// @brief Get all entries.
    const std::unordered_map<std::string, BundledModelEntry>& entries() const;

private:
    std::unordered_map<std::string, BundledModelEntry> entries_;
};

} // namespace entropic::config
```

**Registry YAML structure** (`bundled_models.yaml`):

```yaml

  name: Qwen3.5-35B-A3B-UD-IQ3_XXS
  url: https://huggingface.co/unsloth/Qwen3.5-35B-A3B-GGUF/resolve/main/...
  size_gb: 13.1
  adapter: qwen35
  description: Primary workhorse

router:
  name: Qwen3-0.6B-Q8_0
  url: ...
  size_gb: 0.6
  adapter: qwen3
```

Parsing uses the same `extract()` helpers from Phase 1:

```cpp
std::string BundledModels::load(const std::filesystem::path& path) {
    auto content = read_file(path);
    if (content.empty()) return "cannot read " + path.string();

    ryml::Tree tree = ryml::parse_in_arena(
        ryml::to_csubstr(path.string()),
        ryml::to_csubstr(content));
    ryml::ConstNodeRef root = tree.rootref();

    for (auto child : root) {
        BundledModelEntry entry;
        entry.key = std::string(child.key().str, child.key().len);
        extract(child, "name", entry.name);
        extract(child, "url", entry.url);
        extract(child, "size_gb", entry.size_gb);
        extract(child, "adapter", entry.adapter);
        extract(child, "description", entry.description);

        if (entry.name.empty()) {
            return "bundled model '" + entry.key + "' missing 'name'";
        }
        entries_[entry.key] = std::move(entry);
    }
    return "";
}
```

### Phase 3: Config Structs -- Populate from YAML

v1.8.0 defined the config structs in `include/entropic/types/config.h`
with C++ default initializers. v1.8.1 adds `parse()` functions that
populate structs from ryml nodes.

`include/entropic/config/loader.h`:

```cpp
/// @file loader.h
/// @brief Config loader -- YAML to C++ structs with validation.

#pragma once

#include <entropic/types/config.h>
#include <entropic/config/bundled_models.h>
#include <filesystem>
#include <string>

namespace entropic::config {

/// @brief Full parsed config result.
///
/// Aggregates all config sections. This is the C++ equivalent of
/// Python's LibraryConfig (engine-only fields, no TUI).
struct ParsedConfig {
    ModelsConfig models;            ///< Tiers + router
    RoutingConfig routing;          ///< Routing rules
    GenerationConfig generation;    ///< Default generation params
    PermissionsConfig permissions;  ///< Tool permissions
    MCPConfig mcp;                  ///< MCP server settings
    CompactionConfig compaction;    ///< Auto-compaction settings
    std::string log_level = "INFO"; ///< Log level string

    /// Constitution: nullopt = bundled default, disabled = explicit false
    std::optional<std::filesystem::path> constitution;
    bool constitution_disabled = false;

    /// App context: nullopt = disabled, path = custom
    std::optional<std::filesystem::path> app_context;
    bool app_context_disabled = false;

    bool inject_model_context = true;  ///< Auto-inject model context
    int vram_reserve_mb = 512;         ///< Reserved VRAM headroom (MB)

    /// Config dir -- base for bundled data discovery.
    std::filesystem::path config_dir;
};

/// @brief Load config using layered resolution.
///
/// Resolution order (highest wins):
/// 1. Compiled defaults (struct initializers)
/// 2. Global config (~/.entropic/config.yaml)
/// 3. Project config (.entropic/config.local.yaml)
/// 4. Environment variables (ENTROPIC_*)
///
/// @param global_path Path to global config file (may not exist).
/// @param project_path Path to project config file (may not exist).
/// @param registry Bundled models registry for path resolution.
/// @param[out] config Output parsed config.
/// @return Empty string on success, error message on failure.
std::string load_config(
    const std::filesystem::path& global_path,
    const std::filesystem::path& project_path,
    const BundledModels& registry,
    ParsedConfig& config);

/// @brief Load config from a single YAML file (no layering).
///
/// Used by entropic_configure_from_file() for single-file configs.
/// Applies compiled defaults first, then overlays the file.
std::string load_config_from_file(
    const std::filesystem::path& path,
    const BundledModels& registry,
    ParsedConfig& config);

} // namespace entropic::config
```

**Parse function pattern:**

```cpp
/// @brief Parse a ModelConfig from a YAML node.
/// @param node YAML node containing model fields.
/// @param registry Bundled models for path resolution.
/// @param[out] config Output model config.
/// @return Empty string on success, error message on failure.
std::string parse_model_config(
    ryml::ConstNodeRef node,
    const BundledModels& registry,
    ModelConfig& config)
{
    // Path resolution: "primary" -> ~/models/gguf/Qwen3.5-35B-...gguf
    std::string path_str;
    if (extract(node, "path", path_str)) {
        config.path = registry.resolve(path_str);
    }

    extract(node, "adapter", config.adapter);
    extract(node, "context_length", config.context_length);
    extract(node, "gpu_layers", config.gpu_layers);
    extract(node, "keep_warm", config.keep_warm);
    extract(node, "use_mlock", config.use_mlock);
    extract(node, "logits_all", config.logits_all);
    extract(node, "reasoning_budget", config.reasoning_budget);
    extract(node, "cache_type_k", config.cache_type_k);
    extract(node, "cache_type_v", config.cache_type_v);
    extract(node, "n_batch", config.n_batch);
    extract(node, "n_threads", config.n_threads);
    extract(node, "tensor_split", config.tensor_split);
    extract(node, "flash_attn", config.flash_attn);
    extract_string_list(node, "allowed_tools", config.allowed_tools_list);

    return "";
}
```

**ModelsConfig parse** -- handles the tiers dict + router + default:

```cpp
std::string parse_models_config(
    ryml::ConstNodeRef node,
    const BundledModels& registry,
    ModelsConfig& config)
{
    // "default" is a special key, not a tier
    extract(node, "default", config.default_tier);

    // "router" is a special key -- ModelConfig, not TierConfig
    if (node.has_child("router")) {
        config.router.emplace();
        auto err = parse_model_config(node["router"], registry, *config.router);
        if (!err.empty()) return "models.router: " + err;
    }

    // All other keys are tier names
    for (auto child : node) {
        std::string key(child.key().str, child.key().len);
        if (key == "default" || key == "router") continue;

        TierConfig tier;
        auto err = parse_tier_config(child, registry, tier);
        if (!err.empty()) return "models." + key + ": " + err;
        config.tiers[key] = std::move(tier);
    }

    return "";
}
```

**Cross-field validation** (replaces Pydantic model_validator):

```cpp
std::string validate_routing(
    const RoutingConfig& routing,
    const ModelsConfig& models)
{
    if (models.tiers.empty()) return "";

    // routing.enabled requires a router model
    if (routing.enabled && !models.router.has_value()) {
        return "routing.enabled is true but models.router is not configured";
    }

    auto err = validate_fallback_tier(routing.fallback_tier, models.tiers);
    if (!err.empty()) return err;

    err = validate_tier_map(routing.tier_map, models.tiers);
    if (!err.empty()) return err;

    err = validate_handoff_rules(routing.handoff_rules, models.tiers);
    if (!err.empty()) return err;

    return "";
}
```

### Phase 6: Layered Config Merge

Config resolution order: compiled defaults -> global YAML -> project YAML -> env vars.

Merge strategy: **overlay, not replace.** A project config that sets only
`models.lead.gpu_layers: 20` overlays that one field -- it does not blow
away the global config's full tier definitions.

Implementation: parse each layer into the SAME `ParsedConfig`, in order.
Since `extract()` only modifies a field when the key is present, absent
keys preserve the value from the previous layer.

```cpp
std::string load_config(
    const std::filesystem::path& global_path,
    const std::filesystem::path& project_path,
    const BundledModels& registry,
    ParsedConfig& config)
{
    // Layer 1: compiled defaults (already in struct initializers)

    // Layer 2: global config
    if (std::filesystem::exists(global_path)) {
        auto err = parse_config_file(global_path, registry, config);
        if (!err.empty()) return "global config: " + err;
    }

    // Layer 3: project config
    if (std::filesystem::exists(project_path)) {
        auto err = parse_config_file(project_path, registry, config);
        if (!err.empty()) return "project config: " + err;
    }

    // Layer 4: environment variable overrides
    apply_env_overrides(config);

    // Validate after all layers merged
    std::vector<std::string> warnings;
    auto err = validate_config(config, warnings);
    for (const auto& w : warnings) {
        spdlog::warn("{}", w);
    }
    return err;
}
```

**Frontmatter parser implementation:**

```cpp
std::string parse_prompt_file(
    const std::filesystem::path& path,
    PromptType expected_type,
    ParsedPrompt& result)
{
    auto content = read_file(path);
    if (content.empty()) {
        return "cannot read prompt file: " + path.string();
    }

    // Must start with "---"
    if (content.substr(0, 3) != "---") {
        return "prompt file " + path.string() + " missing YAML frontmatter";
    }

    // Split on "---" delimiter: "", yaml_block, body
    auto second_delim = content.find("---", 3);
    if (second_delim == std::string::npos) {
        return "prompt file " + path.string() + " has malformed frontmatter";
    }

    std::string yaml_block = content.substr(3, second_delim - 3);
    std::string body = content.substr(second_delim + 3);

    // Parse YAML frontmatter
    ryml::Tree tree = ryml::parse_in_arena(
        ryml::to_csubstr(path.string()),
        ryml::to_csubstr(yaml_block));
    ryml::ConstNodeRef root = tree.rootref();

    // Validate type field
    std::string type_str;
    if (!extract(root, "type", type_str)) {
        return "prompt file " + path.string() + " missing 'type' field";
    }

    PromptType actual_type;
    if (type_str == "constitution") actual_type = PromptType::CONSTITUTION;
    else if (type_str == "app_context") actual_type = PromptType::APP_CONTEXT;
    else if (type_str == "identity") actual_type = PromptType::IDENTITY;
    else return "prompt file " + path.string()
                + " has unknown type '" + type_str + "'";

    if (actual_type != expected_type) {
        return "prompt file " + path.string() + " has type '" + type_str
               + "' but was loaded as '"
               + prompt_type_to_string(expected_type) + "'";
    }

    result.type = actual_type;
    extract(root, "version", result.version);

    // Trim leading/trailing whitespace from body
    auto start = body.find_first_not_of(" \t\n\r");
    if (start != std::string::npos) {
        body = body.substr(start);
    }
    auto end = body.find_last_not_of(" \t\n\r");
    if (end != std::string::npos) {
        body = body.substr(0, end + 1);
    }
    result.body = std::move(body);

    return "";
}
```

```cpp
std::string load_identity(
    const std::filesystem::path& path,

{
    ParsedPrompt base;
    auto err = parse_prompt_file(path, PromptType::IDENTITY, base);
    if (!err.empty()) return err;

    identity.body = std::move(base.body);

    // Re-parse frontmatter YAML for identity-specific fields

    auto content = read_file(path);

    ryml::Tree tree = ryml::parse_in_arena(
        ryml::to_csubstr(path.string()),
        ryml::to_csubstr(yaml_block));
    ryml::ConstNodeRef root = tree.rootref();

    auto& fm = identity.frontmatter;
    fm.type = PromptType::IDENTITY;
    extract(root, "version", fm.version);
    extract(root, "name", fm.name);
    extract_string_list(root, "focus", fm.focus);
    extract_string_list(root, "examples", fm.examples);

    // Optional string fields
    std::string grammar_str, auto_chain_str;
    if (extract(root, "grammar", grammar_str))
        fm.grammar = grammar_str;
    if (extract(root, "auto_chain", auto_chain_str))
        fm.auto_chain = auto_chain_str;

    extract_string_list_opt(root, "allowed_tools", fm.allowed_tools);
    extract_string_list_opt(root, "bash_commands", fm.bash_commands);

    extract(root, "max_output_tokens", fm.max_output_tokens);
    extract(root, "temperature", fm.temperature);
    extract(root, "repeat_penalty", fm.repeat_penalty);
    extract(root, "enable_thinking", fm.enable_thinking);
    extract(root, "model_preference", fm.model_preference);
    extract(root, "interstitial", fm.interstitial);
    extract(root, "routable", fm.routable);
    extract(root, "role_type", fm.role_type);
    extract(root, "explicit_completion", fm.explicit_completion);

    // Phases (optional map of named phases)
    if (root.has_child("phases")) {
        fm.phases.emplace();
        for (auto child : root["phases"]) {
            PhaseConfig phase;
            std::string phase_name(child.key().str, child.key().len);
            extract(child, "temperature", phase.temperature);
            extract(child, "max_output_tokens", phase.max_output_tokens);
            extract(child, "enable_thinking", phase.enable_thinking);
            extract(child, "repeat_penalty", phase.repeat_penalty);
            extract_string_list_opt(child, "bash_commands", phase.bash_commands);
            (*fm.phases)[phase_name] = std::move(phase);
        }
    }

    // Benchmark (optional)
    if (root.has_child("benchmark")) {
        fm.benchmark.emplace();
        auto bench_node = root["benchmark"];
        if (bench_node.has_child("prompts")) {
            for (auto p_node : bench_node["prompts"]) {
                BenchmarkPrompt bp;
                extract(p_node, "prompt", bp.prompt);
                // checks stored as raw YAML for now -- consumer parses
                if (p_node.has_child("checks")) {
                    for (auto c_node : p_node["checks"]) {
                        std::string check_yaml;
                        ryml::emitrs_yaml(c_node, &check_yaml);
                        bp.checks_json.push_back(std::move(check_yaml));
                    }
                }
                fm.benchmark->prompts.push_back(std::move(bp));
            }
        }
    }

    // Validate: focus must have at least one entry
    if (fm.focus.empty()) {
        return "identity " + path.string() + ": focus must have at least one entry";
    }
    if (fm.name.empty()) {
        return "identity " + path.string() + ": name must not be empty";
    }

    return "";
}
```

/// @brief Lightweight tier descriptor for classification prompt building.
///
/// Only the fields needed to build the prompt. Not a full identity --
/// avoids coupling the prompt builder to the full identity system.
struct TierDescriptor {
    std::string name;                   ///< Tier name (e.g., "lead")
    std::vector<std::string> focus;     ///< Focus areas
    std::vector<std::string> examples;  ///< Few-shot examples
};

/// @brief Round-robin interleave few-shot examples across tiers.
///
/// Cycling through tiers prevents recency bias -- no single tier's
/// examples dominate the tail of the prompt. The 0.6B router model
/// is sensitive to example ordering; grouped-by-tier ordering caused
/// it to parrot the last tier's digit.
///
/// @param tiers Ordered tiers (index+1 = classification digit).
/// @return Lines formatted as: "example text" -> digit
std::vector<std::string> interleave_examples(
    const std::vector<TierDescriptor>& tiers);

/// @brief Auto-generate classification prompt from tier focus + examples.
///
/// Output format:
/// ```
/// Classify the message. Reply with the number only.
///
/// 1 = LEAD: triage incoming requests, ...
/// 2 = ENG: write code, ...
///
/// "example1" -> 1
/// "example2" -> 2
/// "user message" ->
/// ```
///
/// The trailing " -> " (with space) constrains the router model to
/// output a single digit. Used with max_tokens=1 for classification.
///
/// @param tiers Ordered tiers (index+1 = classification digit).
/// @param message User message to classify.
/// @param history Recent user messages for context (optional).
/// @param recent_tiers Recent tier activations for continuity (optional).
/// @return Classification prompt string.
std::string build_classification_prompt(
    const std::vector<TierDescriptor>& tiers,
    const std::string& message,
    const std::vector<std::string>& history = {},
    const std::vector<std::string>& recent_tiers = {});

**Implementation:**

```cpp
std::vector<std::string> interleave_examples(
    const std::vector<TierDescriptor>& tiers)
{
    // Build per-tier example queues: (digit, example_text)
    struct Queue {
        int digit;
        std::vector<std::string> examples;
    };

    std::vector<Queue> queues;
    for (size_t i = 0; i < tiers.size(); ++i) {
        if (!tiers[i].examples.empty()) {
            queues.push_back({static_cast<int>(i + 1), tiers[i].examples});
        }
    }

    std::vector<std::string> lines;
    size_t round_idx = 0;
    while (true) {
        bool added = false;
        for (const auto& q : queues) {
            if (round_idx < q.examples.size()) {
                lines.push_back("\"" + q.examples[round_idx] + "\" -> "
                                + std::to_string(q.digit));
                added = true;
            }
        }
        if (!added) break;
        ++round_idx;
    }
    return lines;
}

std::string build_classification_prompt(
    const std::vector<TierDescriptor>& tiers,
    const std::string& message,
    const std::vector<std::string>& history,
    const std::vector<std::string>& recent_tiers)
{
    std::string result;
    result += "Classify the message. Reply with the number only.\n\n";

    // Tier descriptions from focus
    for (size_t i = 0; i < tiers.size(); ++i) {
        result += std::to_string(i + 1) + " = " + to_upper(tiers[i].name) + ": ";
        for (size_t j = 0; j < tiers[i].focus.size(); ++j) {
            if (j > 0) result += ", ";
            result += tiers[i].focus[j];
        }
        result += "\n";
    }
    result += "\n";

    // Tier continuity context
    if (!recent_tiers.empty()) {
        result += "Recent tiers: ";
        for (size_t i = 0; i < recent_tiers.size(); ++i) {
            if (i > 0) result += " -> ";
            result += recent_tiers[i];
        }
        result += "\n\n";
    }

    // History context (compact for small router model)
    if (!history.empty()) {
        result += "Recent messages: ";
        size_t start = history.size() > 5 ? history.size() - 5 : 0;
        for (size_t i = start; i < history.size(); ++i) {
            if (i > start) result += " | ";
            result += history[i];
        }
        result += "\n\n";
    }

    // Interleaved examples
    auto example_lines = interleave_examples(tiers);
    for (const auto& line : example_lines) {
        result += line + "\n";
    }

    // Final line: message to classify (trailing space for digit output)
    result += "\"" + message + "\" -> ";

    return result;
}
```

```cpp
extern "C" entropic_error_t entropic_configure_from_file(
    entropic_handle_t handle,
    const char* config_path)
{
    if (!handle || !config_path) {
        return ENTROPIC_ERROR_INVALID_ARGUMENT;
    }

    try {
        entropic::config::BundledModels registry;
        auto data_dir = entropic::config::resolve_data_dir_from_path(config_path);
        auto err = registry.load(data_dir / "bundled_models.yaml");
        if (!err.empty()) {
            spdlog::error("Failed to load bundled models: {}", err);
            // set_last_error(handle, err);
            return ENTROPIC_ERROR_INVALID_CONFIG;
        }

        entropic::config::ParsedConfig config;
        err = entropic::config::load_config_from_file(
            std::filesystem::path(config_path), registry, config);
        if (!err.empty()) {
            spdlog::error("Config validation failed: {}", err);
            // set_last_error(handle, err);
            return ENTROPIC_ERROR_INVALID_CONFIG;
        }

        // Store config on handle (handle internals defined in v1.8.4)
        // handle->config = std::move(config);

        spdlog::info("Config loaded from {}", config_path);
        return ENTROPIC_OK;

    } catch (const std::exception& e) {
        spdlog::error("Config load exception: {}", e.what());
        // set_last_error(handle, e.what());
        return ENTROPIC_ERROR_INTERNAL;
```
