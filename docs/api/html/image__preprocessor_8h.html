Detailed Description

Image preprocessing for multimodal inference.

Validates, decodes, resizes, and outputs pixel data ready for the vision encoder. Uses stb_image (bundled with llama.cpp) for decoding.

Thread safety
Stateless after construction. Multiple threads can call preprocess methods concurrently on different images.
