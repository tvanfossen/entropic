## Scope

Wire the C API facade (`entropic.h`) to the fully-implemented C++
engine internals. 62 functions declared, 54 are stubs. The engine
handle struct does not exist — every subsystem is built and tested
but unreachable from the public API.

v2.0.0 is the point where the engine becomes a shippable library.
After this version, consumers can link against `librentropic.so`,
call `entropic_create()`, configure, run, and destroy. The Python
wrapper (auto-generated from `entropic.h`) wraps the same C API.

**Not in scope:** examples porting (v2.0.1), doxygen traceability
(v2.0.2), TUI port (post-2.0.0), fine-tuning pipeline (v2.1.0).
